1. What is the difference between var, let, and const?
Ans: The difference between var, let, and const is that var is function-scoped. It is redeclared and reassigned. let is block-scoped and cannot be redeclared in the same scope. Also, it can be reassigned. Const is also block-scoped but it can't be redeclared or reassigned.
2. What is the spread operator (...)?
Ans: The spread operator in JavaScript is used to spread elements of an array or object. It helps to copy, merge, or pass values easily.
3. What is the difference between map(), filter(), and forEach()?
Ans: map(), filter(), and forEach() are JavaScript array methods used to loop in arrays, but they are used in different cases. Map() creates a new array by transforming every element of the original array. filter() creates a new array that matches a condition. forEach() loops through an array but does NOT return a new array.

Ans: Template literals are a feature in JavaScript that allows for the insertion of variables, writing multi-line strings, and dynamic expressions
